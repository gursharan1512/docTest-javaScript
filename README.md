# docTest-javaScript

This project compares the two document on the basis of Wordcount, NounCount, VerbCount, AdjectiveCount and Similarity and gives output reading from json file which includes all the information about the result.


## Built With

* [JavaScript]
* [nodejs] (https://nodejs.org/en/)
* [Sublime text 3](https://www.sublimetext.com/3)
